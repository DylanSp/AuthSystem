DTOs:

ResourceDetailsDTO
	Value (string)

ResourceDTO
	ID (Guid)
	Details (ResourceDetailsDTO)

UserAuthenticationDTO
	Username (string)
	Password (string)

UserPasswordDTO
	Password (string)

PermissionGrantDetailsDTO
	ResourceID (Guid)
	Username (string)
	PermissionType (string)

PermissionGrantDTO
	PermissionGrantID (Guid)
	Details (PermissionGrantDetailsDTO)



Endpoints:

AuthController
	/login POST
		takes UserAuthenticationDTO, returns cookie
	/logout POST
		takes nothing, returns nothing

UserController
	/users POST
		takes UserAuthenticationDTO, returns UserAuthenticationDTO
	/users/{username} PUT
		takes UserPasswordDTO, returns nothing
	/users/{username} DELETE
		takes nothing, returns nothing

ResourceController
	/resources GET
		takes nothing, returns List<ResourceDTO>
	/resources POST
		takes ResourceDetailsDTO, returns ResourceDTO
	/resources/{resourceId} GET
		takes nothing, returns ResourceDTO
	/resources/{resourceId} PUT
		takes ResourceDetailsDTO, returns ResourceDTO
	/resources/{resourceId} DELETE
		takes nothing, returns nothing

PermissionsController
	/permissions POST
		takes PermissionGrantDetailsDTO, returns PermissionGrantDTO
	/permissions/{permissionId} DELETE
		takes nothing, returns nothing

	/resources/{resourceId}/permissions GET
		takes nothing, returns List<PermissionGrantDTO>
